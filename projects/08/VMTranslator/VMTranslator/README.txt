Usage:
1. Make sure you have add the location of /bin folder in CLASSPATH in Windows environment variable.
2. open command window (cd to where your .vm file (or directory with .vm file in it) located)
3. copy the "implement" folder to current directory
4. type: java VMTtranslator xxx.vm or xxx(directory name)
5. process will be displayed in command window 
6. program generates the "xxx.asm" in the same folder
7. done

Note. "implement" composed of the implementation of each vm command.
